🔧 Configuration — JSONBin.io

Suis ces étapes une seule fois pour connecter l'admin au site client :

  1. Va sur jsonbin.io → connecte-toi à ton compte
  2. Clique sur Bins dans le menu → Create a Bin
    Contenu : [] → Nom : stars-reservations → Save
  3. Crée un 2ème bin → Contenu : {} → Nom : stars-price → Save
  4. Pour chaque bin, copie son BIN ID (visible dans l'URL ou dans le panneau)
  5. Va dans API Keys → copie ta Master Key (commence par $2a$10$...)

🔒 Tes clés restent uniquement dans ce navigateur (localStorage). Elles ne sont jamais envoyées ailleurs.
